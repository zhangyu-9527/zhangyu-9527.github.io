cpp学习文档（二）

2.1 数据类型

cpp规定了在定义数据名称变量或者常量的时候必须指定数据类型
type name=inital;
数据类型存在的意义，给变量分配合适的内存空间
（不同类型占据不同大小的内存空间）

2.2 整型

超出范围之后报错，回归最小值

不同整形类型 占据内存大小字节 取值范围
short 2 ±2^15
int 4 ± 2 ^31
long 4 ± 2^31
longlong 8 ±2^63

2.3 sizeof

利用sizeof求出数据类型占用内存大小
sizeof（short）

2.4 实型（浮点数）

意义：表示小数
浮点型小数表示两种：1. 单精度小数float 2. 多精度小数double
默认情况下会展示6位有效数字
统计数据类型所占内存空间 sizeof（float）
其他：科学计数法 float f=3e2; 3*10^2

数据类型 占用字节 有效数字范围
float 4 7位有效数字
double 8 15-16位有效数字

2.5 字符型变量

意义：字符型变量用于显示单个字符
char ch=’d’，创建字符型变量只能用单引号
占用1个字节大小
字符型变量不是将本身放到内存中而是ASCII编码放到储存单元里
（int）ch//ASCII码强转为整形 A 65 A 97

2.6 转义字符

换行符 \n
输出特殊符号 \ print\
水平制表符 \t ——整齐的输出数据

2.7 字符串

用于表示一串字符

  1. char str[]=”hello world” 1. [] 2. “” //c++字符串
  2. string str =”hello world” //cpp风格字母串 需要包含头文件 #include

2.8布尔类型bool

定义：布尔类型数据代表真或者假
只有两个值 true 真(1) false 假(0)
bool占用一个字节大小

2.9数据的输入cin

cin>>变量
布尔类型非0都等于1
