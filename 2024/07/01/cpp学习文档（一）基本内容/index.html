cpp学习文档（一）

一、hello world

注意

  1. cpp源文件中必须有且只能出现一个main函数，main函数是程序的一个入口
    可通过注释解决/* */
  2. iostream 输入输出数据流，包含数据缓存区
  3. 注释单行注释、多行注释
  4. // 详细信息 /注释内容/
  5. endl 换行并清空数据缓存区

二、单双行注释

// 详细信息 单行注释
/注释内容/ 多行注释

三、变量的意义

定义：给一段指定的内存空间起名
意义：方便管理内存空间
方法：1. 内存地址 2.内存命名 type（数据类型） 数据名=变量初始值； int a=10;

四、常量的意义

程序中不可改变的数值

  1. #define 宏常量 通常在程序顶部 #define 常量名 常量值
  2. const 变量定义前加const const 数据类型 常量名=初始数值

五、关键字

程序中变量中不可用的变量名

六、标识符号命名规则

字母、数字、下划线
首字母不能是数字，可以是字母与下划线
标识符号区分大小写
命名标识符号的时候最好可以见名知意的作用
